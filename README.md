# Детекция пустот на полках

Проект представляет собой систему на базе компьютерного зрения для автоматического определения пустых мест на полках магазинов. Он использует нейронную сеть (архитектура U-Net), чтобы анализировать фотографии полок и сообщать, если товар необходимо пополнить.

## Структура проекта

* `AI.py` — Скрипт для обучения модели семантической сегментации. Включает пайплайн подготовки данных (аугментация с помощью Albumentations), инициализацию модели U-Net с энкодером ResNet34, функции обучения и валидации (с подсчетом метрик: IoU, Precision, Recall, F1).
* `UI_gradio.py` — Основной веб-интерфейс на базе **Gradio**. Позволяет загрузить фотографию полки и наглядно увидеть результат с выделенной красным цветом маской пустот.
* `UI.py` — Альтернативный веб-интерфейс на базе **Streamlit**. Выполняет ту же самую задачу, предоставляя аналогичный функционал.
* `best_model.pth` — Обученные веса модели (появляются после успешного завершения обучения или скачиваются отдельно).
* `requirements.txt` — Список всех необходимых зависимостей для работы проекта.

## Требования

Для запуска проекта убедитесь, что у вас установлен Python (рекомендуется 3.8+, протестировано на 3.13) и создано виртуальное окружение. Вы можете установить все нужные библиотеки с помощью команды:

```bash
pip install -r requirements.txt
```

*(Рекомендуется использовать среду с поддержкой CUDA для ускорения вычислений и обучения)*.

## Использование

### 1. Запуск веб-интерфейса (Gradio)
Если у вас уже есть обученная модель `best_model.pth`, вы можете запустить Gradio-приложение через обычный Python. Это рекомендуемый способ:

```bash
python UI_gradio.py
```
После запуска перейдите по локальной ссылке `http://127.0.0.1:7860/` в вашем браузере.

### 2. Запуск альтернативного веб-интерфейса (Streamlit)
Если вы хотите использовать старую версию интерфейса на Streamlit, для его запуска используйте встроенную команду фреймворка (а не обычный `python`):

```bash
streamlit run UI.py
```

### 3. Обучение модели
Если вы хотите обучить модель на своих данных:
1. Подготовьте датасет. Данные должны быть разбиты на папки `Train`, `Val`, и `Test`, внутри которых находятся папки `images` (фотографии) и `masks` (черно-белые маски пустот).
2. В файле `AI.py` измените путь к датасету в переменной `DATASET_PATH`.
3. Запустите скрипт обучения:
```bash
python AI.py
```
В процессе обучения лучшая модель (по метрике IoU) будет автоматически сохранена в файл `best_model.pth`.

## Принцип работы
* **Модель:** U-Net с предобученным на ImageNet энкодером ResNet34.
* **Функции потерь:** Комбинация Dice Loss и Binary Cross Entropy (BCEWithLogitsLoss).
* **Обработка в UI:** Загружаемое изображение приводится к размеру 256x256, нормализуется, после чего модель предсказывает маску пустот. Эта маска масштабируется обратно до исходного размера и накладывается поверх оригинального изображения полупрозрачным красным цветом.
